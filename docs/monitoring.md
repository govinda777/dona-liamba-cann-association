# Monitoramento e Observabilidade

## Vercel Analytics

Utilizamos o Vercel Analytics para monitorar métricas de performance (Core Web Vitals) e acessos em tempo real.

## Sentry (Futuro)

Planejamos integrar o Sentry para rastreamento de erros e logs detalhados em produção.

## Alertas

Configure alertas na Vercel para falhas de build e erros de runtime críticos.
